# Teslo API

1. Clonar proyecto
2. Ejecutar el comando 
```
yarn install
```
3. Cloanr el arcivo ```.env.template ``` y renombrarlo a ``` .env```
4. Cambair las variables de entorno
5. Levantar la base de datos
```
docker-compose -d
```
6. Ejecutar en modo desarrollo
```
yarn start:dev
```

6. Ejecutar SEED
```
http//localhost:3000/api/seed
```
